Project Methodology 📊

Step-by-step process followed to build the Food Delivery Time Prediction system.

1. Data Collection 📁

Collected dataset containing delivery details such as distance, weather, traffic, vehicle type, and delivery time.

2. Data Preprocessing 🧹

Handled missing values, cleaned data, and converted categorical variables into numerical format using encoding techniques.

3. Feature Engineering ⚙️

Selected important features like distance, preparation time, traffic, and experience to improve prediction accuracy.

4. Model Building 🤖

Applied Random Forest Regression algorithm to train the model for predicting delivery time.

5. Model Evaluation 📈

Evaluated model performance using metrics like MAE and R² score to ensure accuracy.

6. Model Saving 💾

Saved trained model and scaler using joblib for reuse in the web application.

7. Web Development 🌐

Designed frontend using HTML, CSS, and Bootstrap to collect user inputs.

8. Backend Integration 🔗

Used Flask framework to connect frontend with the trained ML model.

9. Deployment 🚀

Deployed the application to allow users to predict delivery time through a web interface.
